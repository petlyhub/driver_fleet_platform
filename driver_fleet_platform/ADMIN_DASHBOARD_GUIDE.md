# 📊 لوحة التحكم الإدارية - دليل الاستخدام

## الميزات المضافة

### 1. 🔐 صفحة تسجيل الدخول المحمية
- كلمة مرور بسيطة للحماية (افتراضية: `admin123`)
- جلسة صالحة لمدة 24 ساعة
- يمكن تغيير كلمة المرور عبر متغير البيئة `ADMIN_PASSWORD`

### 2. 📈 الإحصائيات السريعة
- إجمالي عدد السائقين
- عدد السائقين قيد المراجعة
- عدد السائقين المقبولين
- عدد السائقين المرفوضين

### 3. 👥 جدول السائقين
عرض جميع السائقين مع المعلومات التالية:
- الاسم الكامل
- البريد الإلكتروني
- رقم الجوال
- المدينة
- نوع المركبة
- تاريخ التسجيل
- الحالة الحالية

### 4. 🎯 فلترة السائقين
يمكنك فلترة السائقين حسب:
- الكل
- قيد المراجعة
- المقبولون
- المرفوضون

### 5. ⚙️ أزرار التحكم بالحالة
- ✓ قبول السائق
- ✗ رفض السائق
- 👁 تحويل إلى "تحت المراجعة"

## المسارات الجديدة في Backend

| المسار | الطريقة | الوصف |
|--------|---------|-------|
| `/api/admin/login` | POST | تسجيل دخول المسؤول |
| `/api/admin/logout` | POST | تسجيل خروج المسؤول |
| `/api/admin/stats` | GET | جلب الإحصائيات |
| `/api/drivers` | GET | جلب جميع السائقين (محمي) |
| `/api/drivers/:id/status` | PUT | تحديث حالة سائق (محمي) |

## كيفية الاستخدام

### 1. الوصول للوحة التحكم
1. افتح التطبيق في المتصفح
2. اضغط على زر "🔐 دخول الإدارة" في الأعلى
3. أدخل كلمة المرور (الافتراضية: `admin123`)
4. اضغط على "تسجيل الدخول"

### 2. إدارة السائقين
- عرض جميع السائقين في الجدول
- استخدام الفلاتر لعرض حالات معينة
- الضغط على ✓ لقبول السائق
- الضغط على ✗ لرفض السائق
- الضغط على 👁 لتحويل الحالة إلى "تحت المراجعة"

### 3. تغيير كلمة المرور
في ملف `.env`:
```env
ADMIN_PASSWORD=كلمة_المرور_الجديدة
```

## الأمان

⚠️ **مهم**: 
- كلمة المرور الافتراضية هي `admin123` - يجب تغييرها في الإنتاج!
- الجلسات تنتهي بعد 24 ساعة
- في الإنتاج، يُنصح باستخدام نظام مصادقة أكثر أماناً
